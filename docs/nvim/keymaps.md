# Keymaps

Leader key: `Space`

## Navigation

| Mode | Key | Action |
|------|-----|--------|
| Normal | `<Esc>` | Clear search highlights |
| Normal | `<C-h/j/k/l>` | Navigate windows |
| Normal | `<C-d>` / `<C-u>` | Scroll half-page, cursor centered |
| Visual | `J` / `K` | Move selected lines down / up |
