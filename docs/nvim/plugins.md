# Plugins

| Plugin | Purpose |
|--------|---------|
| [Telescope](telescope/) | Fuzzy finder for files, grep, git branches, LSP symbols |
| [LSP](lsp/) | Language server (go-to-def, hover, diagnostics) |
| [nvim-tree](nvim-tree/) | File explorer sidebar |
| Catppuccin | Colorscheme (Mocha flavour, transparent background) |
| gitsigns | Inline git hunk signs in gutter; stage/reset/preview hunks |
| diffview | VS Code-style diff panel and file history |
| Harpoon 2 | Fast file navigation via named slots |
| Neogit | Git UI in a floating window; integrates with diffview + Telescope |
| Neotest | In-editor test runner; pytest adapter via `uv run python` |
| Noice | Floating cmdline, message routing, notification popups |
| lualine | Statusline: mode, branch, diagnostics, filename, location |
| nvim-treesitter | Syntax highlighting, treesitter-based folding |
